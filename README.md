<h1 align="center">Laravel API Base Project</h1>

## About this project

 This is a Laravel API Base that help to start new project within short time. In this project used ORM, Jobs, Service, Facade, Passport, Sanctum, Gate, Middleware, Module, BaseController and Others.

Laravel is accessible, powerful, and provides tools required for large, robust applications.

## How to setup

Step 1: Create a new .env file and copy .env.example <br>
Step 2: Create a Database following the .env file Database name
